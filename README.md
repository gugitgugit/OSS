## 접속 방법
#### 1. VSCode에서 Terminal을 연다.
#### 2. 해당 레포지토리를 git clone 한다.
```
$ git clone https://github.com/gugitgugit/OSS.git
```
#### 3. `project` 디렉토리로 들어간다.
#### 4. 해당 프로젝트에서 사용한 라이브러리를 다운받는다.
```
$ npm install .
```
#### 5. 리액트를 실행한다.
```
$ npm start
```

## 로그인 방법
#### 해당 서비스에서는 `회원가입` 기능을 별도로 제공하지 않습니다.  
ID : 구준혁  
PW : password  
#### 위 ID, PW를 입력하시면 로그인이 됩니다.
